##About project

This is an ongoing project where I'm going through and familiarize myself with Blazor. The project is based on Blazor
webassembly template as foundation. Pages, classes and services has been added to create a prototype for educational
planning platform. It is inspired by a system tried developed at NTNU (Norwegian university of science and technology) 
for planning and monitor resources connected to education of students at different fields. 

This project has focus on mainly front end of the system. Therefore it has not been connected to a database. 
To populate and test the system it has a couple of testclasses with hardcoded data. 

When handling data this system follows the principles for CRUD. 

Components folder contains most of the pages, classes, lists, interfaces and services for this project
